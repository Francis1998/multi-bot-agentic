# Release Notes — v0.4.24

**Date:** 2026-03-23
**Repository:** multi-bot-agentic

## Highlights

This release focuses on stability and developer-experience improvements
to the `multi_bot_agentic` core.

## Changes

### Added
- `rationale` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed event trace output
- Unit tests for ODA boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running rationale worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average event latency by ~12% via connection pooling
- Cleaner error messages for ODA validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade multi-bot-agentic
```
