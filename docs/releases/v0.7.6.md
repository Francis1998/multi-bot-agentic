# Release Notes — v0.7.6

**Date:** 2024-10-18
**Repository:** multi-bot-agentic

## Highlights

This release focuses on stability and developer-experience improvements
to the `multi_bot_agentic` core.

## Changes

### Added
- `bot` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed ODA trace output
- Unit tests for orchestrator boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running bot worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average ODA latency by ~12% via connection pooling
- Cleaner error messages for orchestrator validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade multi-bot-agentic
```
